## üí° Key Concepts Quick Reference

### **What's a URL-Based QR Code?**
A QR code that contains a direct URL (like `https://yourdomain.com/qr/room/form/1/`) instead of encoded data.

**Benefits**: Works with any smartphone camera, no special app needed.

### **What are Public Forms?**
Forms accessible without login at URLs like `/qr/room/form/<pg_id>/`

**Purpose**: Allow anyone with the QR code to add rooms/residents directly.

### **What's Different From Old System?**
- Old: QR encoded room/resident data (required custom scanner)
- New: QR contains form URL (works with any phone camera)

### **Is It Secure?**
Yes! URLs are context-specific, no sensitive data in QR, standard CSRF protection.

### **Do I Need to Regenerate Old QR Codes?**
Yes, old data-based QR codes won't work with the new system. You can regenerate them by visiting the QR code pages.
